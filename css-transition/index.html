关于 Transition Timing Function

ease-out：动画开始快，逐渐减慢，适合退出效果。

ease：默认值，开始和结束稍慢，中间较快。

linear：匀速运动，无加速或减速。

ease-in：动画开始慢，逐渐加速，适合进入效果。

steps：动画分步进行，steps(10, end)表示分10步完成，结束时跳跃。

ease-in-out：开始和结束都慢，中间快，平滑过渡。

将鼠标悬停在方块上，观察不同过渡效果的动画差异！
